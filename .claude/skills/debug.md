# Debug

When debugging:

1. **Reproduce** — Confirm you understand the exact failure (error message, stack trace, steps to reproduce).
2. **Isolate** — Narrow down where in the code the failure originates.
3. **Hypothesize** — State the most likely root causes before diving in.
4. **Verify** — Read the relevant code. Check assumptions against actual state.
5. **Fix** — Make the minimal change that resolves the root cause. Don't patch symptoms.
6. **Confirm** — Suggest how to verify the fix works.
