# Plan

Run the full planning sequence for a new initiative.

## Instructions

Run in order — do not skip steps:

**Step 1 — Product direction review**: Challenge what we're building. Confirm the right problem is being solved. Validate scope, use cases, and whether this is the real user need.

**Step 2 — Engineering design**: Define architecture, data flow, diagrams, edge cases, failure modes, and test coverage.

**Step 3 — Delegate**
- Hand product use cases and spec to `product-manager`
- Hand architecture decisions to `architect` for ADR
- Hand UX requirements to `ux-designer` for component specs
- Open a sprint entry in `.claude/context/sprint.md` via `scrum-master`

Do not begin implementation until all three steps are complete.
